Massive Wallhack
================
TKTL:n kurssien aikataulut -> Google Calendar

* Kurssien listaus
* Kurssin luentojen / laskareiden valinta
* icalendar-tiedoston export
* Käyttäjien hallinta
* Käyttäjän kalenteritietojen muistaminen
* Kalenterin esikatselu
